Elementos de un sistema de informacion computacional

Equipo Computacional

Hardware necesario para que el sistema de informacion funcione, integrado por computadoras y equipo periferico conectado a ellas

Recurso Humano

Son las personas que interactuan con el sistema de informacion, suministrar datos o utilizan el sistema para generar datos

Fuente

Son los datos e informacion que se introducen en el sistema, es decir, son las entradas requeridas para que el sistema informatico funcione.

Programas

Software ejecutado por la computadora, el cual procesa la informacion de entrada paragenerar los resultados esperados.

Telecomunicaciones

Abarca el software y hardware que permite la transmision en forma electronica de texto e imagenes, entre otros.

Politicas y Reglas de Operacion

Aplicadas a los procedimientos y mecanismos para hacer que un aplicacion de computadora trabaje de acuerdo con estas o regir la parte funcional del proceso administativo de la empresa.
